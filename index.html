HTML & CSS CheatSheet

HTML Tags Reference

html5 declaration

<!DOCTYPE html>

The declaration will be at the first line of any html document.

html

<html>...</html>

This will contain html structure and contents.

head

<head>...</head>

It contains metadata for a web page. It's used for page indexing. All the links of stylesheets,fonts and the title tag will be here.

title

<title>...</title>

web page name in the browser's tab.

body

<body<...</body>

contains All type of contents which is shown on a web page

header

<header>...</header>

A container usually for navigational links and introductory contents.

main

<main>...</main>

A container for the main contents of the web page.

footer

<footer>...</footer>

A container for the subject information at the bottom of the web page.

navs

<nav>...</nav>

A block of navigational links such as menus and table of contents.

headings

<h1..h6>...</h1..h6>

Six different heading tags for a variety of purposes.<h1>..</h1> is used for main headings.All others are for subheading.

divs

<div>...</div>

a container that divides the page into sections. It's very useful for grouping elements.

Paragraph

<p>...</p>

contains a block of plain text.

short pieces of text

<span>...</span>

contains short pieces of text that are inline.

emphasize text

<em>...</em>

render the text as italic.

Important text

<strong>...</strong>

render the text as bold.

line breaks

<br>

produce a line break in text. It's an empty tag.

unordered lists

<ul> <li>...</li> <li>...</li> </ul>

create a list of items in no particular order with a bullet point.

Ordered list

<ol> <li>...</li> <li>...</li> </ol>

create a list of items where each item is numbered.

images

<img>

insert images. It has no ending tag.

videos

<video>...<

insert videos.

Audio

<audio>...</audio>

insert audios.

links

<a>...</a>

an anchor tag which will add a link.

Comments

<!--...-->

add comments to the code and the content between the tags will be ignored by the browser.

Performatted text

<pre>...</pre>

defines performatted text.The text will be displayed exactly as written in html source code.

Figure tag

<figure>...</figure>

a block element used to designate self-contained content that does not affect the main flow of the article

figure caption tag

<figcaption>...</figcaption>

add a caption to describe the media in figure tag.

Aside element

<aside>...</aside>

represents a portion of a document whose content is only indirectly related to the document's main content such as side-bar.

article tag

<article>...</article>

hold contents like articles,blogs and so on.

section tag

<section>...</section>

defines elements in a document such as chapters, headings or any other area of the document.
